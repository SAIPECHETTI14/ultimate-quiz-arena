# Ultimate Quiz Arena

A modern advanced quiz web application built using:

- HTML5
- CSS3
- Vanilla JavaScript
- Open Trivia Database API
- LocalStorage

---

# Features

## Home Page

- Animated glassmorphism UI
- Dark/Light mode
- Responsive navbar
- Mobile support
- Animated gradients
- Category selection
- Difficulty selection

## Quiz System

- API-based quiz questions
- Offline custom categories
- Timer system
- Progress bar
- Sound effects
- Auto next question
- Keyboard shortcuts
- Animated answer feedback

## Result System

- Score percentage
- Confetti animation
- Motivational messages
- Share results
- Time tracking

## Leaderboard

- LocalStorage leaderboard
- Auto sorting
- Daily challenges
- Achievements system

## Offline Categories

- Coding
- Science
- Anime
- Movies
- Sports

# How To Run

## VS Code

1. Open VS Code
2. Open quiz-game folder
3. Install Live Server extension
4. Right click index.html
5. Click "Open with Live Server"

---

# Deployments

## GitHub Pages

1. Create GitHub repository
2. Upload all files
3. Go to Settings
4. Pages
5. Select branch main
6. Save

Your site will go live.

---

## Netlify

1. Open Netlify
2. Drag and drop quiz-game folder
3. Deploy

---

## Vercel

1. Import GitHub repository
2. Click deploy
